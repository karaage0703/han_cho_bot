# han_cho_bot
Dice game of betting Han(odd) or cho(even) 

## Dependency

- Python3
- macOS / Linux / JetPack(Jetson Nano)
- TensorFlow==2.x
- myCobot Atom version 2.7
- pymycobot==2.3.3

## Setup

```sh
$ pip3 install pymycobot==2.3.3
```
